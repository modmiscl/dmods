HTML Page Structure

Below is a visualization of an HTML page structure:

<html></html>
<head></head>
<title>Page title</title>
<body></body>
<h1>I am a heading</h1>
<p>I am a paragraph</p>
<p>I am another paragraph</p>

Note: Only the content inside the <body> section (the white green above) is displayed in a browser.

Shadow DOM and Imports

ShadowDOM

Styles must be in the template tags.

dream-garage.html


      <template>
        <style>
          h1 { color: green; }
        </style>
        <h1>Hello from Shadow Root</h1>
      </template>
      <script>
        var template = document.querySelector('template');
        var host = document.getElementById('host');
        var root = host.createShadowRoot();
        root.appendChild(document.importNode(template.content, true));
      </script>

Insertion Point

Distributed nodes aren't actually inserted into insert point, they merely are displayed there.

<content> creates insertion point, written like this anything not already inserted will be. Elements inserted are called distributed nodes.

<content select="img"> means insert images here.

insertion-point.html


      <comment-form>
        <h1>Tell us what's on your mind.</h1>
        <p class="instructions">Fill out this form.</p>
        <p class="footer">Privacy Policy</p>
      </comment-form>


      <template>
        <fieldset>
          <legend>
            <content select="h1"></content>
          </legend>
          <content select="p.instructions"></content>
          <textarea style="width:400px;height:200px;"><textarea>
          <br />
          <input type="submit" value="Submit Comment">
          <content select="p.footer"></content>
        </fieldset>
      </template>
      <script>
      var CommentProto = Object.create(HTMLElement.prototype);
      CommentProto.createCallback = function() {
        var template = document.querySelector('template');
        var shadow = this.createShadowRoot();
        shadow.appendChild(document.importNode(template.content, true));
      }
      document.registerElement('comment-form', {
        prototype: CommentProto
      });
      </script>

Import

Link to HTML file must be inside <Head> tag, the CSS and Javascript properties execute immediately, but the HTML isn't until called.

dream-garage.html


      <ul>
        <li>Nissan</li>
        <li>Lamborghini</li>
        <li>Audi</li>
      </ul>

Option 1:


      <!DOCTYPE html>
      <head>
        <link rel="import" href="dream-garage.html" id="dream-garage">
      </head>
      <body>
      <script>
        var content = document.querySelector('#dream-garage').import;
        var carList = content.querySelector('ul');
        document.body.appendChild(carList);
        </script>
      </body>

Option 2:


      <!DOCTYPE html>
      <head>
        <script>
          var link = document.createElement('link');
          link.rel = 'import';
          link.href = 'dream-garage.html';
          link.id = 'dream-garage';
          document.head.appendChild(link);
        </script>
      </head>
      <body></body>

Import File in ShadowDOM

Must use "document.currentScript.ownerDocument" because if "document.querySelector" was used it would search the file that's importing not the original file that is being imported.

index.html


      <!DOCTYPE html>
      <head>
        <link rel="import" href="dream-garage.html" id="dream-garage">
      </head>
      <body>
        <dream-garage></dream-garage>
      </body>

dream-garage.html


        <template>
          <a href="#" id="view-disclaimer">Disclaimer</a>
          <ul>
            <li>Nissan</li>
            <li>Lamborghini</li>
            <li>Audi</li>
          </ul>
        </template>
        <script>
        (function(){
        var Proto = Object.create(HTMLElement.prototype);
        var importDoc = document.currentScript.ownerDocument;
        Proto.createCallback = function() {
          var template = importDoc.querySelector('template');
          var templateClone = document.importNode(template.content, true);
          var disclaimer = templateClone.querySelector('#view-disclaimer');
          disclaimer.addEventListener('click', function() {
            alert('I can only get one.');
          });
          var root = this.createShadowRoot();
          root.appendChild(templateClone);
        };
        document.registerElement('dream-garage', {
          prototype: Proto
        })();
        </script>

Handling Events

Use "onload" and "onerror" and <script> tag must come before the file is imported.


      <script>
      function loaded(event) {
        console.log('Import Loaded');
      }
      function error(event) {
        console.log('Couldn't load '+event.target.href);
      }
      </script>
      <link rel="import" href="import.html" onload="loaded(event)" onerror="error(event)">
